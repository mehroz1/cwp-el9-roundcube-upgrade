# Roundcube Upgrade for Control Web Panel (CWP) on EL9


This script is tested on EL9 even works with old centos 7 installations but due to PHP limitation (php package bundled with cwp) only 1.4.x can be insatalled on centos 7 and only 1.5.x and 1.6.x works with EL9

# EL9

Rouncube Versions: 1.5.x, 1.6.x

Tested Version: 1.6.15

Tested OS: Alma linux 9

Shipped PHP version: 7.4

CWP version: 0.9.8.1224


# CentOS 7

Roundcube version: 1.4.x

Tested version: 1.4.15

Tested OS: CentOS 7

Shipped PHP Version: 7.1 or 7.2

CWP version: 0.9.8.1224

# Note

Adjust the version numbers in script to the required ones according to your OS before using the script. I will not claim responsiblity for any damages. Also I haven't test the script with Alma linux 8 so I can't if this will work for el8 or not.
